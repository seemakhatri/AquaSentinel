# AquaSentinel

ML-powered groundwater depletion prediction system using Python, Machine Learning, and Government groundwater datasets.

## Features
- Groundwater trend analysis
- Feature engineering
- Random Forest prediction
- Data visualization
- Real government dataset integration

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Dataset
Government of India Atal Bhujal Yojana groundwater dataset.

## Outputs
- Groundwater trend graphs
- District comparison charts
- ML-based groundwater risk prediction
